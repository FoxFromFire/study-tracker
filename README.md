
# ⏱ Трекер учёбы

Простая консольная программа на Python для отслеживания времени, потраченного на учёбу. Позволяет запускать таймер сессий, сохранять историю и считать общее время занятий.

## 📌 Возможности

- 🟢 Старт / 🔴 Стоп таймера учебной сессии
- 📋 Просмотр истории всех сессий
- 🗑 Очистка истории
- ⏳ Подсчёт общего времени занятий
- 💾 Автоматическое сохранение данных в `sessions.json`

## 🚀 Установка и запуск

1. Убедитесь, что установлен Python 3.7+
2. Скачайте файлы `main.py` и `timer.py` в одну папку
3. Запустите программу:

```bash
python main.py
```

## 🕹 Использование

При запуске откроется меню:
=== Трекер учёбы ===

⏸ Таймер остановлен

Старт
История
Выход


- **1** — запускает или останавливает таймер (в зависимости от текущего состояния)
- **2** — открывает подменю истории:
  - `1` — показать историю сессий
  - `2` — удалить всю историю
  - `3` — показать суммарное время
  - `4` — вернуться в главное меню
- **3** — выход из программы

## 📁 Структура проекта
.

├── main.py          # точка входа, главное меню

├── timer.py         # логика таймера и работы с историей

└── sessions.json    # файл с сохранёнными сессиями (создаётся автоматически)

## 💾 Формат данных

Каждая сессия сохраняется в `sessions.json` в формате:

```json
{
    "start": "14:32:10",
    "end": "15:10:45",
    "duration": 38
}
```

## 🛠 Технологии

- Python 3
